# Quickom Link API Documentation

This repository contains Markdown documentation for the Quickom Link alias API.

## Base URL

```text
https://link.quickom.edu.vn
```

## Authentication

API requests use API key authentication.

```http
Authorization: <api_key>
```

| Document                           | Description                          |

| [Create Alias](docs/create.md)     | Create a new alias.                  |
| [Retrieve Alias](docs/retrieve.md) | Retrieve an alias by ID.             |

All endpoint documentation is written in Markdown and includes request details, parameters, examples, and response formats.
