# Quickom Link API Documentation

This repository contains Markdown documentation for the Quickom Link alias API.

## Base URL

```text
https://link-mytv.quickom.com
```

## Authentication

API requests use bearer token authentication.

```http
Authorization: Bearer <your_token>
```
